# Exercises

- [x] Make five programs, one for each basic arithmetic operation. Your program must have two variables, a and b, defined at the beginning with the values ​​to be operated. Make programs for:
    Addition
    Subtraction
    Multiplication
    Division
    Module
- [x] Make a program that returns the greater of two numbers. Define two variables at the beginning of the program with the values ​​to be compared.
- [x] Make a program that returns the largest of three numbers. Define three variables at the beginning of the program with the values ​​to be compared.
- [ ] Make a program that, given a value defined in a variable, returns "positive" if that value is positive, "negative" if it is negative and "zero" otherwise.
- [ ] Make a program that defines three variables with the values ​​of the three internal angles of a triangle. Return true if the angles represent the angles of a triangle and false otherwise.
- [ ] For the angles to be a valid triangle, the sum of the three must be 180 degrees. If the attracted angles are invalid, the program should return an error message.
- [ ] Write a program that is named after a chess piece and returns the movements it makes.
As a challenge, make the program work both if you receive the name of a part with uppercase letters and with lowercase letters, without increasing the number of conditionals.
As a tip, you can search for a function that makes a string have all lowercase (lowercase) letters.
If the part passed to invalid, the program should return an error message.
Example: bishop -> diagonals
- [ ] Write a program that converts a grade given in percentage (from 0 to 100) into concepts from A to F. Follow these rules:
Percentage> = 90 -> A
Percentage> = 80 -> B
Percentage> = 70 -> C
Percentage> = 60 -> D
Percentage> = 50 -> E
Percentage <50 -> F
The program should return an error message and end if the note is passed below 0 or above 100.
- [ ] Write a program that defines three numbers in variables at the beginning and returns true if at least one of the three is even. Otherwise, it returns false.
Bonus: use only one if.
- [ ] Write a program that defines three numbers in variables at the beginning and returns true if at least one of the three is odd. Otherwise, it returns false.
Bonus: use only one if.

Your program should also issue an error message and terminate if any of its input values ​​are less than zero.
